## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- Modern web browser
- Operating system: Windows, macOS, or Linux

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/local-games.git
cd local-games
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```
